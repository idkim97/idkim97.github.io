## 2. 로컬 개발 환경

### 필수 도구
- rbenv (Homebrew로 설치됨)
- Ruby 3.2.8 (`rbenv local 3.2.8` 설정됨)
- Bundler (`gem install bundler`)

### 서버 실행 명령어
```bash
cd /Users/yeonsu/project/idkim97.github.io
RUBYOPT="-EUTF-8" bundle exec jekyll serve
```
- 접속: http://127.0.0.1:4000
- `RUBYOPT="-EUTF-8"` 필수 (한글 주석 인코딩 이슈)

### rbenv이 안 먹힐 때
```bash
eval "$(rbenv init - zsh)"
```
- `~/.zshrc`에 추가되어 있으므로 새 터미널에서는 자동 적용

## 4. 주의사항

### SCSS 인코딩
- SCSS 파일에 한글 주석 사용 시 파일 상단에 `@charset "utf-8";` 필수
- 빌드 시 `RUBYOPT="-EUTF-8"` 환경변수 필요
- 적용된 파일: `_variables.scss`, `_base.scss`, `_sidebar.scss`, `_navigation.scss`, `elements.scss`, `layout.scss`, `minimal-mistakes.scss`, `variables.scss`, susy vendor 파일들

### 브레이크포인트 (반응형 기준)
```
$small: 600px
$medium: 768px
$medium-wide: 900px
$large: 1024px
$x-large: 1280px
```

### 사이드바 너비 (커스터마이징됨)
```
$right-sidebar-width-narrow: 100px (기본 150px에서 변경)
$right-sidebar-width: 300px
$right-sidebar-width-wide: 350px
```

## 5. 변경 이력

### 2025-06-12 | 구글 애드센스 자동 광고 표시 오류 수정

**문제**: 블로그 글 우측에 광고가 중간에 나타났다가 스크롤 시 오른쪽으로 밀려나며 사라짐

**원인**:
- `_variables.scss`의 `.adsense-left` CSS (`position: fixed`)가 자동 광고에 영향
- `_base.scss`의 글로벌 `transition: all 0.2s`가 광고 요소에도 적용되어 위치 이동 시 애니메이션 발생

**수정**:
- `_variables.scss`: 사용하지 않는 `.adsense-left` 스타일 삭제
- `_base.scss`: `ins.adsbygoogle` 요소에 `transition: none !important` 추가

**참고 - 현재 광고 구조**:
- `footer/custom.html`에서 AdSense 스크립트 로드 (Auto Ads)
- `single.html`, `default.html`의 수동 광고는 모두 주석 처리 상태
- AdSense 계정: `ca-pub-5634079691781840`

### 2025-06-12 | 모바일 반응형 수정 + 로컬 개발환경 구축

**문제**: 모바일에서 화면이 깨짐

**원인 및 수정 내용**:

| 파일 | 문제 | 수정 |
|------|------|------|
| `_page.scss` | `#main`에 `margin-left: 12em` 고정 → 모바일에서 콘텐츠가 오른쪽으로 밀림 | 모바일: `margin-left: auto` + 양쪽 `padding: 1em`, `$large` 이상에서만 `margin-left: 12em` |
| `_reset.scss` | `.greedy-nav .site-title img` `width: 400px` 고정 → 모바일에서 넘침 | 600px 이하에서 `width: 250px; height: auto` |
| `_sidebar.scss` | `.author__avatar img` `200px × 200px` 고정 → 모바일에서 너무 큼 | 모바일: `80px × 80px`, `$large` 이상: `200px × 200px` |

**개발환경 구축**:
- rbenv + Ruby 3.2.8 설치
- `bundle install`로 gem 의존성 설치
- SCSS 한글 인코딩 이슈 해결 (`@charset "utf-8"` 추가)
