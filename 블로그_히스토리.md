### 2025-06-12 | 구글 애드센스 자동 광고 표시 오류 수정

**문제**: 블로그 글 우측에 광고가 중간에 나타났다가 스크롤 시 오른쪽으로 밀려나며 사라짐

**원인**:
- `_variables.scss`의 `.adsense-left` CSS (`position: fixed`)가 자동 광고에 영향
- `_base.scss`의 글로벌 `transition: all 0.2s`가 광고 요소에도 적용되어 위치 이동 시 애니메이션 발생

**수정**:
- `_variables.scss`: 사용하지 않는 `.adsense-left` 스타일 삭제
- `_base.scss`: `ins.adsbygoogle` 요소에 `transition: none !important` 추가

**참고 - 현재 광고 구조**:
- `footer/custom.html`에서 AdSense 스크립트 로드 (Auto Ads)
- `single.html`, `default.html`의 수동 광고는 모두 주석 처리 상태
- AdSense 계정: `ca-pub-5634079691781840`
